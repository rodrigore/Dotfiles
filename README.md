## [Rodri](http://twitter.com/rodri_gore) dotfiles

- Operative System: [MacOS](https://www.apple.com/macos/)
- Terminal: [Wezterm](https://wezfurlong.org/wezterm)
- MacOS package manager: [Homebrew](https://brew.sh)
